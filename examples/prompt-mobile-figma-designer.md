# Example: create Figma MVP screens

Use this prompt in Claude Code after installing the plugin and connecting Figma MCP.

```text
/mobile-mvp-product:mobile-figma-designer

Using this Figma file: <FIGMA_FILE_URL>
Create a mobile MVP design for an iOS app called AI Plant Care.

Core idea:
- User adds a plant by camera/photo
- App identifies likely plant and creates a care plan
- App helps track watering, light, problems, and recovery
- Avoid looking like a generic Plant Identifier clone

Create:
1. minimal design system
2. onboarding
3. home with plant care tasks
4. add plant / scan screen
5. AI result screen
6. plant care plan screen
7. problem diagnosis screen
8. paywall
9. settings
10. empty/loading/error states

Use iOS-style layout, clean MVP design, auto layout, reusable components, and clear layer names.
```
